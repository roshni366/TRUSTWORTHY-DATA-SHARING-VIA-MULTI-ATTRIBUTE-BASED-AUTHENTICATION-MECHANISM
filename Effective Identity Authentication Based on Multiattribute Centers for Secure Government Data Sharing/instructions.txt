
Blockchain installation instructions

install python 3.7.2 and i did all settings inside hello-eth ethereum tool


install node-v12.13.1-64.msi software and this software available inside code folder
then install packages from requirements.txt file

i did all setup just follow screens shots


blockchain server can be start from hello-eth folder and server starting details are already given in screenshots

After contract deployment in etherum black window you will get contract address under 'Authentication' section and that contract
address you replace in IdentityAuthentication/AuthenticationApp/views.py python file

in IdentityAuthentication/AuthenticationApp/views.py file you can see contract address like '0xd374Cb05bd6187D6cF905D7bBD85f2b704fBDD29' 
this at line no 33 and just replace this with new contract address

python application can be started from IdentityAuthentication/runServer.bat 
just double click on 'runServer.bat' file to start python application and follow screen shots
